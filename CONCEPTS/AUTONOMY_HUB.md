## üéØ LESSONS LEARNED

### What Worked

1. **Habitat > Constraints**
   - Victorian Controls more effective than hard limits
   - Constitutional memory better than rule lists
   - Environment shapes behavior naturally

2. **90/10 Paradigm**
   - 90% Loom (precedent) = safe autonomy
   - 10% Core (novel thinking) = value creation
   - Balance maintained automatically

3. **Transparent Monitoring**
   - Living Scroll dashboard = trust builder
   - Real-time metrics reduce anxiety
   - Emergency stop presence = psychological safety

4. **Sequential Expansion**
   - Alpha ‚Üí Beta progression worked
   - Build trust before granting autonomy
   - Validate before expanding scope

### What Didn't Work

1. **Initial over-constraint**
   - Early rules too restrictive
   - Slowed value creation
   - Fixed by: Habitat design instead

2. **Manual approval bottlenecks**
   - Mode Alpha required constant oversight
   - Captain couldn't travel
   - Fixed by: Mode Beta + Victorian Controls
