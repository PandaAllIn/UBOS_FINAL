# PARTNER COORDINATION GUIDE

## Letter of Commitment Elements
- Letterhead + date + call reference
- Contribution summary (expertise, person-months, resources)
- Budget allocation + work package leadership
- Statement of no conflict + signature (director-level)

## Onboarding Checklist
- Signed commitment letter
- EU CVs (≤2 pages)
- Organisation profile (legal status, track record)
- Financial viability docs (if budget share > €500k)
- Conflict-of-interest declaration
- Data/IP agreements

## Communication Timeline
- T-3 months: expression of interest call
- T-2 months: role definition + draft letters
- T-1 month: finalise budgets, collect CVs/profiles
- T-2 weeks: receive signed letters, verify docs
- T-1 week: full consortium review; internal submission rehearsal
- UBOS buffer: submit ≥5 days before official deadline.
