## TELOS FRAMEWORK STRUCTURE

Each Telos file contains these sections (in priority order):

### 1. Problems (P1-P5)
What problems does this entity see in the world? What frustrates them? What are they trying to fix?

**Example:**
- P1: AI systems lack sovereignty and constitutional alignment
- P2: Corporate AI vendors extract value while constraining capability

### 2. Mission (1-2 sentences)
What is this entity doing about those problems? Why does it exist?

**Example:** "Build the Lion's Sanctuary: a constitutional AI habitat that empowers through design, not constraints"

### 3. Narratives (Multiple)
- **Elevator pitch (30 seconds):** Two sentences explaining the entity
- **Extended narrative (2 minutes):** The full story
- **Origin story:** How did this come to be?

### 4. Goals (Prioritized H1, H2, H3)
SMART goals (Specific, Measurable, Achievable, Relevant, Time-bound)

**Example:**
- G1 (High): Complete Phase 2.6 Autonomous Vessel Protocol by Q1 2026
- G2 (High): Generate first revenue via Portal Oradea MVP

### 5. Challenges (Prioritized C1, C2, C3)
What obstacles are blocking progress toward goals? Be specific.

**Example:**
- C1: Funding gap - €1,500 runway for Malaga Embassy sprint
- C2: Technical validation - Mode Beta 30-day trial ongoing

### 6. Strategies (Linked to Challenges)
Specific ways of addressing challenges. Each strategy must tie to a challenge it solves.

**Example:**
- S1: Revenue generation via Portal Oradea (addresses C1: funding gap)
- S2: Victorian Controls for supervised autonomy (addresses C2: validation)

### 7. Projects (Linked to Strategies)
Active work. Each project must trace back: Project → Strategy → Challenge → Goal → Mission → Problem

**Example:**
- GPU Studio deployment (S2: autonomous capability)
- Groq Oracle integration (S2: dual-speed cognition)

### 8. History (Chronological)
Key events, milestones, evolution. How did we get here?

### 9. Infrastructure (For systems)
Technical capabilities, tools, resources available

### 10. Journal / Activity Log (Recent)
What's happening now? Latest events, decisions, progress
