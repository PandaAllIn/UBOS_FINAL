AUTOMATION NOTES FOR DEVELOPERS
================================================================================

**If Implementing via API:**

```python
import requests
import datetime

PERPLEXITY_API_KEY = "your_key_here"
BASE_URL = "https://api.perplexity.ai/chat/completions"

daily_queries = [
    "European Commission funding opportunities November 2025 new calls announced",
    "Horizon Europe 2026 calls opening announcement latest news",
    # ... (all 7 daily queries)
]

def run_daily_scan():
    results = []
    for query in daily_queries:
        response = requests.post(
            BASE_URL,
            headers={"Authorization": f"Bearer {PERPLEXITY_API_KEY}"},
            json={
                "model": "llama-3.1-sonar-large-128k-online",
                "messages": [{"role": "user", "content": query}],
                "return_citations": True,
                "search_recency_filter": "day"  # Last 24 hours
            }
        )
        results.append(response.json())
    return results

# Run at 06:00 UTC daily via cron
```
